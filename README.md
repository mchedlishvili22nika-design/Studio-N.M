# Studio-N.M

Studio-N.M is a modern and responsive web agency landing page built with HTML, CSS, and JavaScript.

The website is designed for a fictional web studio that offers Web Design, Frontend Development, and SEO services. It includes a clean layout, pricing packages, portfolio section, FAQ accordion, and a contact form UI.

## Features

- Modern agency landing page
- Responsive design for desktop, tablet, and mobile
- Web Design, Frontend, and SEO service sections
- Pricing package cards
- Portfolio/demo project cards
- Work process section
- FAQ accordion
- Contact form with basic validation
- Clean and professional UI
